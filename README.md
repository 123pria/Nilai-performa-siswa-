## Deskripsi Project
Project ini bertujuan untuk menganalisis **faktor-faktor yang mempengaruhi performa akademik siswa** berdasarkan dataset *Students Performance*.  
Analisis difokuskan pada hubungan antara **nilai ujian** dengan **gender, tingkat pendidikan orang tua, jenis makan siang, serta persiapan ujian**.

## Tujuan Analisis
1. Mengetahui faktor apa saja yang mempengaruhi nilai siswa
2. Membandingkan performa siswa berdasarkan gender
3. Menganalisis dampak test preparation terhadap nilai ujian
4. Mengidentifikasi pola dan insight dari data pendidikan

---

## Dataset
- Nama dataset: `StudentsPerformance.csv`
- Jumlah data: 1000 siswa
- Fitur utama:
  - Gender
  - Parental level of education
  - Lunch
  - Test preparation course
  - Math score
  - Reading score
  - Writing score

---

## Tahapan Analisis
1. **Data Understanding**
   - Melihat struktur data
   - Mengecek tipe data dan ukuran dataset

2. **Data Cleaning**
   - Mengecek missing value
   - Mengecek data duplikat
   - Validasi nilai numerik

3. **Exploratory Data Analysis (EDA)**
   - Analisis distribusi nilai
   - Perbandingan nilai berdasarkan kategori
   - Analisis korelasi antar nilai ujian

4. **Visualization**
   - Bar chart
   - Boxplot
   - Histogram
   - Correlation heatmap

---

## Insight Utama
- Siswa yang mengikuti **test preparation course** memiliki nilai rata-rata lebih tinggi
- **Pendidikan orang tua** berpengaruh terhadap performa akademik siswa
- Terdapat korelasi positif antara **nilai membaca, menulis, dan matematika**
- Perbedaan performa berdasarkan gender terlihat pada beberapa mata pelajaran
